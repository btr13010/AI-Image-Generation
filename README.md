# Image Generation App with OpenAI Dall-E API

## Technologies

- **Front-end**: React.js, Vite, Tailwind

- **Back-end**: Express.js 

- **Database**: MongoDB 

- **Model**: OpenAI Dall-E 

- **Cloud Storage**: Cloudinary
